# Code of Conduct (Contributor Covenant v2.1)

We pledge to make participation a harassment-free experience for everyone.

Unacceptable behavior includes harassment, discrimination, hate speech, doxxing,
and sustained disruption of discussions.

## Scope
All project spaces and when representing the project.

## Reporting
Report privately via GitHub Security Advisories or by email listed in
SECURITY.md. Do not open public issues for conduct or security incidents.

## Enforcement
Actions may include: correction, warning, temporary ban, permanent ban.
Enforcement will be fair and proportionate.
