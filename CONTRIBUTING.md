# Contributing

Thank you for contributing to Project Wordline.

- Sign commits: `git commit -s` (DCO 1.1).
- Code files use SPDX headers:
  - Apache-2.0 for code
  - MIT-0 for examples/ci/devnet
- Docs carry: "(c) 2025 Lewej Whitelow (lewejahlil) - CC BY 4.0".

## Dev setup
- Rust: rustup + cargo
- Solidity: Foundry (forge, cast)
- Nix (optional): reproducible builds

## Commits
- Prefer conventional commits.
- Include tests for behavior changes.

## Security
- See SECURITY.md. Use GitHub Security Advisories for vulnerability reports.
