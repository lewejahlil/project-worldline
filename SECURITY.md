# Security Policy

## Supported
Until the first stable release, only branch `main` is supported.

## Reporting a vulnerability
- Preferred: GitHub Security Advisory (private).
- Alternate: email lewejahlil+security@users.noreply.github.com
Do not file public issues for vulnerabilities.

We acknowledge within 3 business days and aim for a remediation plan or status
within 14 days. Default coordinated disclosure window: 90 days.

## Out of scope
Denial-of-service without a specific exploit path, third-party dependency issues
without a repro through this project, social engineering, or physical attacks.

## Safe harbor
Good-faith security research following this policy will not be pursued legally.
