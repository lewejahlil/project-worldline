# DEVELOPMENT ZKEY — single contributor — NOT production grade

The `worldline_stf_final.zkey` in this directory was generated with a
single-contributor Phase 2 ceremony for development and testing purposes only.

**Before any mainnet or public testnet deployment**, a multi-party computation
(MPC) ceremony with multiple independent contributors is required to ensure
the toxic waste (tau) is properly destroyed.

See `docs/security/zk-ceremony.md` for ceremony requirements.
